NEADY FILM Portfolio Demo v2
============================

이 버전은 큰 타이틀과 섹션명은 영어로 유지하고, 본문/버튼/설명/서비스/프로세스 문구를 한국어 중심으로 수정한 포트폴리오 데모입니다.

파일 구성
---------
index.html      메인 웹페이지
style.css       디자인/반응형 레이아웃
script.js       모바일 메뉴, 작품 필터, 연도 자동 표기
assets/         임시 히어로 이미지와 프로젝트 썸네일

미리 보는 방법
--------------
1. ZIP 파일 압축을 풉니다.
2. index.html을 브라우저로 엽니다.
3. 더미 텍스트, 이미지, 링크를 실제 자료로 교체합니다.

실제 자료 교체 시 필요한 것
---------------------------
각 프로젝트별:
- 프로젝트 제목
- 연도
- 카테고리
- 영상 링크
- 짧은 설명
- 역할
- 클라이언트/아티스트명
- 썸네일 이미지, 권장 16:9

사이트 공통:
- 메인 히어로 이미지 또는 쇼릴 스틸
- 실제 이메일
- 인스타그램 링크
- 유튜브 링크
- 공개 가능한 클라이언트/협업 리스트

배포
----
Netlify, GitHub Pages, Cafe24, 일반 웹호스팅 등 정적 웹사이트를 지원하는 곳에 업로드할 수 있습니다.
